# FOSSASIA Summit 2026 Participant Guide

This is the website for participant guide, detailing questions regarding event, travel and navigation for international travelers.

# Contributing

1. Fork the repository and clone
```sh
git clone https://github.com/<yourusername>/fossasia-summit-guide
cd fossasia-summit-guide
```
2. Install dependencies (use yarn over npm or pnpm)
```sh
yarn install
```
3. Run the development server. Website should be live at `http://localhost:5173`
```sh
yarn docs:dev
```
4. Make necessary changes in `guide` folder where the content is present.
5. Build and preview the website at `http://localhost:4173`
```sh
yarn docs:build
yarn docs:preview
```
6. Raise a PR to the main repository with necessary changes after successful build and preview.
